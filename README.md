
# Approximate-Pi

This project implements a approximation of pi in several different programming languages.  
This is done via [Summing a Circle's Area](https://en.wikipedia.org/wiki/Approximations_of_%CF%80#Summing_a_circle's_area) which roughly places random points on a plane and checks if they are inside or outside of that circle. An unofficial goal of this project is to have all the letters of the alphabet represented in the initial letters of the [programming language](https://en.wikipedia.org/wiki/List_of_programming_languages#E) **(15/26)**.

## Disclaimer
Those implementations are neither comparable nor should they show benchmarks of any programming language the algorithm was implemented with. I try to keep the implementation as simple as possible. The files might not satisfy best practices of the used programming language - improvements suggestions are welcome though.
